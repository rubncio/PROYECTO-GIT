Git es un sistema de control de versiones distribuido que se utiliza para rastrear los cambios en el código fuente de un proyecto, permitiendo a los desarrolladores trabajar en colaboración de forma segura. Su propósito principal es gestionar el historial de modificaciones, facilitar la vuelta a versiones anteriores y permitir que múltiples personas trabajen en el mismo proyecto sin que sus cambios entren en conflicto. Es una herramienta fundamental en el desarrollo de software por su flexibilidad, velocidad y capacidad para manejar proyectos de cualquier tamaño.  

Un archivo README de Git es un archivo de documentación que describe el proyecto de un repositorio
